# Tic Tac Toe Game

This repository contains a simple Tic Tac Toe game implemented in Python, provided in a Jupyter Notebook (`Tic Tac Toe game.ipynb`). The game is played on an 8-square grid.

## Table of Contents

- [Introduction](#introduction)
- [Features](#features)
- [Technologies](#technologies)
- [Installation](#installation)
- [Usage](#usage)

## Introduction

The `Tic Tac Toe game.ipynb` notebook contains a straightforward implementation of the classic Tic Tac Toe game. Players take turns to mark a square on an 8-square grid, aiming to get three in a row to win the game.

## Features

- Classic Tic Tac Toe gameplay on an 8-square grid.
- Simple and intuitive user interface.
- Written entirely in Python.

## Technologies

- **Python**: The primary language used for implementing the game.
- **Jupyter Notebook**: Provides an interactive environment to play the game.

## Installation

To play the game, you need to have Python installed on your machine. Follow these steps to set up the environment:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/tic_tac_toe.git
    cd tic_tac_toe
    ```
